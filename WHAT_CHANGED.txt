## Troubleshooting:

**App won't start:**
- Check you're in the right directory
- Verify all .pkl files are present
- Try: `pip install fpdf2 openpyxl` if missing

**CSV upload fails:**
- Column names must match template exactly (case-sensitive)
- Check categorical values match allowed options

**Sidebar shows but pages blank:**
- Check for Python errors in terminal
- Verify app.py copied correctly

**Want to revert:**
```bash
cp app_old.py app.py
streamlit run app.py
```
